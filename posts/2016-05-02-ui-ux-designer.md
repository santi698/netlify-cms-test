&nbsp;

<div>
  <strong>Objetivo del puesto / Responsabilidades:</strong>
</div>

<div>
  <ul>
    <li>
      Colaborar con otras areas de Increase en la definición de la experiencia de un producto, teniendo como objetivo lograr la mejor experiencia para los usuarios.
    </li>
  </ul>
</div>

<div>
  <strong>Perfil requerido:</strong>
</div>

<div>
  <ul>
    <li>
      Empatía y saber utilizar insights de pruebas de usuario, research, negocio, mejores prácticas y análisis de competencia para informar decisiones de diseño.
    </li>
    <li>
      Uso de herramientas de diseño de interfaz y prototipado avanzado (Adobe XD o similares).
    </li>
    <li>
      Producir mapas y flujos de información/navegación (mindmaps, flowcharts, sitemaps, etc).
    </li>
    <li>
      Conocimiento y comprensión de plataformas web y mobile, sus estándares y patrones.
    </li>
    <li>
      Independencia y autonomía a la hora de tomar decisiones de diseño.
    </li>
    <li>
      Contar con una web propia o portfolio de proyectos es importante para la postulación.
    </li>
  </ul>
</div>
